# DungeonGame
Dungeon Spel för skol projekt "DV1718 VT26 lp4 Spelprojekt med spelmotor"

## Story
Din son blev fången under krig. 
Efter ett års resande hittade du var han är fången. 
Rescue din son och återvänd överlevande.

## Game Flow
Dungeon crawler där du tar dig genom 3 layers. "City", "Dungeon", "Temple". 
Varje steg spelas ett random event. Monster, Loot, Trap, Nothing.
